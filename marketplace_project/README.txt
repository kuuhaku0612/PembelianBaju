Cara Install Project Marketplace:
1. Jalankan XAMPP, aktifkan Apache & MySQL.
2. Buat database bernama `marketplace_db` di phpMyAdmin.
3. Import file `marketplace.sql` untuk struktur tabel.
4. Buka project di browser via http://localhost/marketplace/
5. Login dan testing fitur.

Folder:
- /admin -> Dashboard Admin
- /penjual -> Penjual tambah produk
- /pelanggan -> Pelanggan lihat produk
- /includes -> File koneksi database
